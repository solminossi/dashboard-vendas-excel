Sobre o Projeto

Este projeto consiste no desenvolvimento de um dashboard interativo em Excel para análise de desempenho 
de vendas de uma operação de delivery fictícia.

O objetivo foi transformar dados brutos em informações visuais claras e acionáveis, permitindo uma leitura 
rápida dos principais indicadores do negócio.


Objetivos da Análise

O dashboard foi estruturado para permitir análise dinâmica de:

- Faturamento total  
- Custos operacionais  
- Lucro líquido  
- Quantidade de vendas  
- Ticket médio  
- Performance por produto  
- Ranking dos Top 3 produtos mais vendidos  


Ferramentas e Recursos Utilizados

- Microsoft Excel  
- Tabelas Dinâmicas  
- Segmentação de Dados (Slicers)  
- Fórmulas Dinâmicas  
- Funções:
  - ÍNDICE  
  - CLASSIFICAR  
  - TEXTO  
- Design de Dashboard e Storytelling Visual  

Principais Análises Implementadas

- Receita por Produto  
- Percentual de Lucro por Produto  
- Comparativo entre Receita, Custo e Lucro  
- Volume de Vendas por Produto  
- Ranking Dinâmico Top 3 de Produtos  

Destaques Técnicos

Um dos diferenciais deste projeto foi a implementação de um ranking dinâmico dos Top 3 
produtos mais vendidos, utilizando fórmulas de classificação e indexação para atualização 
automática conforme os filtros aplicados.

Além disso, o dashboard foi estruturado com foco em:

- Hierarquia visual  
- Clareza na apresentação de KPIs  
- Padronização estética  
- Navegação intuitiva
